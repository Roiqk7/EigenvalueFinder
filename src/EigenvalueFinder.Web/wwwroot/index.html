EigenvalueFinder

?

Enter your matrix elements separated by spaces or commas within each row.

Separate rows using semicolons (;) or newlines (Enter key).

Example for a 2x2 matrix:

1 2; 3 4

Example for a 3x3 matrix:

1, 2, 3
4, 5, 6
7, 8, 9
About

EigenvalueFinder is a tool to find the eigenpairs of a given matrix.

Developed by Roiqk.
